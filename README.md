# Portfolio — Maab Osman

Personal portfolio site built with React, TypeScript, and Vite. Documenting the learning process.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tool & dev server
- **Plain CSS** — custom properties, no framework

## Features

- Bento-grid project cards with live + GitHub links
- Animated hero background (grain texture + cursor spotlight)
- Scroll-reveal animations via IntersectionObserver
- Research log for currently studied topics
- Responsive layout

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
